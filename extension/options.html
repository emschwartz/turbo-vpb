Automatic Text Replacement

TurboVPB will automatically replace the following keywords (including the brackets) in the message templates with the value on the right:

[Your Name]
[Their Name]

New in v0.9.0 Fields from the Additional Info panel in VAN or the About {{Name}} sidebar in OpenVPB can also be automatically inserted. For example, [County] will be replaced with the contact's county if that field is set to be displayed in the phone bank.
