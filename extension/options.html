Congrats on installing TurboVPB!

You'll soon be calling and texting faster than ever!

When you load OpenVPB, there will be a QR Code to scan with your phone. Once your phone is connected, you'll be able to call and (optionally) text contacts with the push of a button.

Use the following settings to enable TurboVPB's easy texting functionality.
(You don't need to configure anything if you only want to make calls.)

Text Message Settings

Create one or more message templates to enable one-click texting.
A button to send each message will appear on TurboVPB on your phone.

Message Templates
Automatic Text Replacement

TurboVPB will automatically replace the following keywords (including the brackets) in the message templates with the value on the right:

[Your Name]
[Their Name]

Enable TurboVPB on Other Sites Beta
